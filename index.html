    Welcome to Kadena Docs¶

    The purpose of this site is to provide a single source for essential Kadena information.

    This documentation is open source and hosted on GitHub which means that anyone can contribute. The goal is to make sure the information is available, clear, concise, and up-to-date.

    Can't find what you're looking for?

    Reach out to the Kadena team on Discord and get support from within any of the dedicated channels

    Public Chain Docs¶

    Run a Node¶

    Resources

    • Chainweb - This takes you to the release page for the Chainweb-node GitHub project. Use this if you want to run your own node.

    Guides

    • Setup a Node - Instructions on how to setup a Kadena node (Community contribution)

    Start mining¶

    Resources

    • BigOlChungus - Open Source Linux AMD/Nvidia OpenCL miner; One instance per card (Community contribution)

    • KDA-Miner - Open Source Linux AMD/Nvidia OpenCL miner; 10% Fee (Community contribution)

    • NoncerPro-Kadena - Closed Source Linux/Windows Nvidia Cuda miner; 2% fee (Community contribution)

    Guides

    • How to Mine Kadena - A medium article by Thanos on how to solo mine Kadena (Community contribution)

    Get a Wallet, Create Keys, & Transfer KDA¶

    Resources

    • Chainweaver - Chainweaver is the official cryptocurrency and smart contract wallet software for the Kadena public blockchain. Currently supports MacOs.

    • Transaction Tester Tool - A simple way to write Pact code and test it out

    • Balance Checker - Allows you to quickly check your balance by inputting a chain address and an account name

    • Bag of Holding - Terminal Wallet (Community contribution)

    • Linux Key Generator - Tools for doing high security key generation (Community contribution)

    • Javascript Transfer Tool - Simple Javascript tools to transfer Kadena coins (Community contribution)

    Guides

    • How to generate a KDA address - A medium article by Thanos on how to Generate a KDA address (Community contribution)

    Write Smart Contracts¶

    • Pact Developer Tutorials - Learn Pact's fundamental concepts through structured lessons, then complete projects to deploy real smart contracts
    • Safe Rotate and Drain example - This example takes an account, rotates it to another key and transfers the whole balance to another account
    • Safe Transfer example - This example does two transfers in a single transaction

    View Network Activity¶

    • Block Explorer - Block Explorer is an analytics tool for the Kadena platform which visualizes the mining, propagation and braiding of blocks across multiple Kadena chains in real time.

    • Active Node List (Community contribution)
